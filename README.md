# Fire-Cursor
# ===========

# ===========
HTML
CSS
Javascript
# ===========

# ===========
Este é um cursor feito com Html, Css e Javascript que reproduz
um efeito flamejante.
Este efeito usa efeito de particulas e uma animação infinita, 
para dar a sensação de que o cursor está em chamas.
